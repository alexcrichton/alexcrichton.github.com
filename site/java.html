Java Applications

Current applications I've written (download and double click jar to run):

  • mds.jar - a drawing program w/ some advanced features
  • calculator.jar - yep, you guessed it
  • ciphers.jar - program supporting encryption and decryption based on various simple ciphers
  • himcm07.jar - ever wondered if you should rent a car?
  • lathe.jar - calculus solids, lots easier
  • mastermind.jar - fun game to play
  • numberspeller.jar - ever wondered how to pronounce 10 3000?
  • pong.jar - old school
  • riemann.jar - Riemann sums anyone?
  • sudoku.jar - that one sudoku you can't solve giving you trouble?
  • tetris.jar - old school and with sound
  • track.jar - if you're a track coach, you might be interested...

All of the source code for these applications can be found on my google code page (check out through SVN). Although I am thinking of moving this to git...

Competitions

In the 2008 Hawkeye Challenge a team of me, Kyle Moose, and Jason Walsh trumped the competiton to get first.

I've written solutions to competitions such as the one in Vermillion and UAB.
The main competition, however, is the USACO. In this difficult, algorithm-based curriculum, I have trudged through solutions to 3/6 chapters, an achievement which is no small feat for me...

As with all applications, all code for these solutions can be found on the google code page in the trunk/java/contests folder.

Litvin

I learned Java mainly from the Internet and my own personal projects, but I was largely helped by the Litvin A & AB books. These two books taught me the rudimentary basics of Java and the more advanced level features of the language.

Yet again, all code written for the books can be found on the google code page... If you're a computer science student, well you just got lucky.

Eclipse
